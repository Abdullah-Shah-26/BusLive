# Contributing to BusLive

Hey there! Thanks for checking out BusLive. We'd love your help making this project better.

## How to Contribute

1. Fork the repo and clone it to your machine
2. Create a new branch: `git checkout -b your-feature-name`
3. Make your changes and test them out
4. Commit with a clear message: `git commit -m "Add: what you did"`
5. Push to your fork: `git push origin your-feature-name`
6. Open a Pull Request and tell us what you built

## Development Setup

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run development server
npm run dev
```

## Code Guidelines

Just keep it simple:
- Follow the existing code style
- Write code that's easy to read
- Test your changes before submitting
- Make small, focused commits

## Need Help?

Stuck on something? Open an issue and we'll figure it out together.
